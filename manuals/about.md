About the "manuals" directory:

## General
This folder has manuals for the tools & instruments in openlab.
* The PDFs are simplified with non-english sections removed
* Additional labels & tags for tools are added to the PDF, or photos of serial numbers
* The manual corresponds to the tool version purchased, or contains notes that highlight the specific version.
* For some manuals, PDFs include the digital manuals found online.  For others, scanned paper copies.

## Format
* The PDFs are compressed for reduced file size.
* Tags are added so the search terms may include many words while filename remains brief
* Relevant sources are noted in manuals, such as the online website for retrieving sister model manuals.
* For scanned pages, the manual is enhanced with OCR - optical character recognition, allowing ctrl-f search for words.
* Bookmarks are added for key pages or sections when the manual is used.

# Updates
* When the manuals are used in the lab, sometimes further notes must be gathered.  These notes become added to the manual PDF as additional pages or as digital comments in the text.
* The name or initials of author are included in important notes, and dates added to comments that have project-based time of edit.
* The files do not contain version numbers - instead, see the metadata containing "date modified."
